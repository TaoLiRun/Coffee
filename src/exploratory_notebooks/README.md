# Exploratory Notebooks

This folder contains Jupyter notebooks for exploratory data analysis.

## Notebooks

- `order.ipynb` - Order Analysis: exploratory analysis of customer order data, including order patterns, distributions, and temporal trends
- `product.ipynb` - Product Analysis: product-level data analysis including product popularity, category distributions, and product lifecycle patterns
- `push.ipynb` - Push Notification Analysis: explores push notification data, including push timing, frequency, and basic distributions of push events
- `push_buy.ipynb` - Push-Purchase Relationship Analysis: examines the relationship between push notifications and purchase behavior

## Usage

```bash
cd scripts/exploratory_notebooks
jupyter notebook
```
